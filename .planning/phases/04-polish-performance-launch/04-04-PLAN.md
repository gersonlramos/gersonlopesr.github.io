After pushing, poll the GitHub Actions run status via the public REST API (no auth token needed for a public repo) every 15 seconds, up to 5 minutes:

    RESP=$(curl -s "https://api.github.com/repos/gersonlramos/gersonlopesr.github.io/actions/runs?branch=main&per_page=1")
    STATUS=$(echo "$RESP" | grep -o '"status":"[a-z_]*"' | head -1)
